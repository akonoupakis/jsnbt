dpd.js

the dpd resource is coming along with the deployd library.

it is widely used by both the admin and the site to perform data access operations.

    <script type="text/javascript" src="dpd.js"></script>

inject the above to the head of your pages pointing to a dynamic "dpd.js" file located in your application root

check the dpd refrence on http://docs.deployd.com/docs/collections/reference/dpd-js.html for a complete guide on dpd.js
