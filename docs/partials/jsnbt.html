jsnbt.js brings an object with all the merged configuration from all installed modules!

This object is meant to deliver static application configuration to both the admin and the site.

<script type="text/javascript" src="jsnbt.js"></script>

Inject the above to the head of your pages pointing to a dynamic "jsnbt.js" file located in your application root or the admin root.
