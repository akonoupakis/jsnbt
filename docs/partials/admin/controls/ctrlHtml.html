an html editor control

ngModel text
the model to be bound to the control
ngDisabled bool
indicates whether this control is disabled or not
ngRequired bool
indicates whether the value of this control is required or not
ngOptions object
options for the native configuration of the tinyMCE editor
ngLabel text
the label for this control, viewed as header
ngTip text
the sublabel for this control, viewed as subheader

usage:

<ctrl-html 
    ng-model="sample.value" 
    ng-disabled="false" 
    ng-required="true" 
    ng-options="{ height: '450px' }"
    ng-label="sample label text" 
    ng-tip="sample tip text"></ctrl-html>
