ngModel array of objects
the model to be bound to the control, including the image src and the cropping generated guidelines
ngDisabled bool
indicates whether this control is disabled or not
ngRequired bool
indicates whether the value of this control is required or not
ngLabel text
the label for this control, viewed as header
ngTip text
the sublabel for this control, viewed as subheader
ngExtensions array
image extensions that are allowed
ngHeight number
the height of the target image
ngWidth number
the width of the target image

usage:

<ctrl-image-list
    ng-model="sample.value" 
    ng-disabled="false" 
    ng-required="true"
    ng-height="300"
    ng-width="600" 
    ng-extensions="['.png', '.jpg', '.jpeg', '.gif', '.tiff']"
    ng-label="sample label text" 
    ng-tip="sample tip text"></ctrl-image-list>
