the node picker control
this control popups a target node tree to select a single node

ngModel objectId
the model to be bound to the control
ngDomain text
the domain to target nodes for
ngOptions object
target options for the node modal selector controller passed in the TreeNodeService retrieval function
ngDisabled bool
indicates whether this control is disabled or not
ngRequired bool
indicates whether the value of this control is required or not
ngLabel text
the label for this control, viewed as header
ngTip text
the sublabel for this control, viewed as subheader

usage:

<ctrl-node 
    ng-model="sample.value"
    ng-domain="core"
    ng-options="{ entities: ['page'] }"
    ng-disabled="false" 
    ng-required="true" 
    ng-label="sample label text" 
    ng-tip="sample tip text"></ctrl-node>
