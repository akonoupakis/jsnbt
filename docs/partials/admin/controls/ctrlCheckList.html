a checkboxlist control

ngModel bool
the model to be bound to the control
ngDisabled bool
indicates whether this control is disabled or not
ngRequired bool
indicates whether the value of this control is required or not (at least one checked checkbox in the list)
ngLabel text
the label for this control, viewed as header
ngTip text
the sublabel for this control, viewed as subheader
ngOptions array of objects
these objects represent the checkboxlist options
ngValueField text
the name of the option property that should be used as value
ngNameField text
the name of the option property that should be used as name
ngDisabledField text
the name of the option property that should be used as disabled
ngDescriptionField text
the name of the option property that should be used as description

usage:

<ctrl-check-list
    ng-model="sample.value" 
    ng-disabled="false" 
    ng-required="false" 
    ng-label="sample label text" 
    ng-tip="sample tip text"
    ng-options="options"
    ng-name-field="name"
    ng-value-field="value"
    ng-disabled-field="disabled"
    ng-description-field="description"></ctrl-check-list>
