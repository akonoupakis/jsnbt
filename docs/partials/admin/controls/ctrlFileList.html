the file list control
this control popups the file manager to select files

ngModel text
the model to be bound to the control
ngDisabled bool
indicates whether this control is disabled or not
ngRequired bool
indicates whether the value of this control is required or not
ngLabel text
the label for this control, viewed as header
ngTip text
the sublabel for this control, viewed as subheader
ngExtensions array
the allowed file extensions

usage:

<ctrl-file-list 
    ng-model="sample.value"
    ng-disabled="false" 
    ng-required="true" 
    ng-label="sample label text" 
    ng-tip="sample tip text"
    ng-extensions="['.pdf']"></ctrl-file-list>
