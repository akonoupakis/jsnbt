ngModel text or number
the model to be bound to the control as the value
ngDisabled bool
indicates whether this control is disabled or not
ngRequired bool
indicates whether the value of this control is required or not
ngLabel text
the label for this control, viewed as header
ngTip text
the sublabel for this control, viewed as subheader
ngOptions array of objects
these objects represent the selectbox options
ngDefault bool
indicates whether a default option value should be included or not
ngValueField text
the name of the option property that should be used as value
ngNameField text
the name of the option property that should be used as name
ngValidate function
a custom validation function that should return a boolean result
ngValid bool
if false, the control returns invalid when validation triggers (so that you can guide its validity from another scope property value)
ngAutoFocus bool
indicates if the control should be focused on load or not

usage:

<ctrl-select
    ng-model="sample.value" 
    ng-disabled="false" 
    ng-required="true" 
    ng-label="sample label text" 
    ng-tip="sample tip text"
    ng-options="options"
    ng-default="true"
    ng-name-field="name"
    ng-value-field="value"></ctrl-select>
