The server object created on application start.

This singleton object exists in the application lifecycle passed among all components, exposing application level properties as the following:

app the app instance that the server deployed on version the jsnbt version of the app running path the path from which the app is running dbg the debug status of the app running environment the environment status of the app running (dev/prod) localization the localization status { enabled: true, locale: 'en' } ssl the ssl status modules the modules loaded into the engine { core: {}, rest: [{}, {}, {}], public: {}, all: [{}, {}, {}, {}, {}] } config the merged configuration of all loaded modules. module configuration will be covered in next chapters. init(options, module) initiates the application app.init({ title: 'custom title' }); createServer(options) creates the server for the application var server = app.createServer({ host: 'localhost:3000', port: 3000, db: { host: 'localhost', port: 27017, name: 'jsnbt' } }); messager the messager engine. it serves the below for mail and sms getTemplate(templateCode, callback) retrieves a template and returns it as an argument on the callback function getModel(templateCode, callback) retrieves the debug model for a template code and returns it as an argument on the callback function parseTemplate(template, model, callback) parses a template with a model and returns the parsed result as an argument on the callback function getSender(dpd, callback) creates a sender and returns it as an argument on the callback function var templateCode = 'registration'; server.messager.mail.getTemplate(templateCode, function (err, tmpl) { if (err) { ctx.error(err); } else { server.messager.mail.getModel(templateCode, function (modelErr, model) { if (modelErr) { ctx.error(modelErr); } else { server.messager.mail.parseTemplate(tmpl, model, function (parseErr, parsedTmpl) { if (parseErr) { ctx.error(parseErr); } else { server.messager.mail.getSender(ctx.dpd, function (senderErr, sender) { if (senderErr) { ctx.error(senderErr); } else { sender.send({ to: 'email@domain.com', subject: parsedTmpl.subject, body: parsedTmpl.body }, function (sendErr, response) { if (sendErr) { ctx.error(sendErr); } else { ctx.json({ sent: true }); } }); } }); } }); } }); } }); cache returns the server cache getPath(path) returns the server path of a relative path start(next) starts the server server.start();
