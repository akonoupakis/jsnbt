The context object created per request.

Commonly, through this object the engine resolves and ends a request. req the request res the response method the request method session the session cookies the cookies user the current user timer the timer timing the request life cycle node the resolved node pointer the resolved pointer node layout the resolved layout meta the meta information for the selected language params the key value collection of meta tag values robots the selected robot values template the template code of the template to be rendered uri the url in parts restricted whether this request is restrited or not error(code, stack) renders an error page and ends the request view() renders the selected template with the current context configuration and ends the request json(data) renders the given json object as json and ends the request html(text) renders the given text as html and ends the request redirect(url, code) redirects to the given url with the given code (or 302 as default)
