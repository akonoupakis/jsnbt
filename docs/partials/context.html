context

The context object created per request.

Commonly, through this object the engine resolves and ends a request.

req object
the request
res object
the response
method string
the request method
session object
the session
cookies object
the cookies
user object
the current user
timer object
the timer timing the request life cycle
node object
the resolved node
pointer object
the resolved pointer node
layout string
the resolved layout
meta object
the meta information for the selected language
params array of objects
the key value collection of meta tag values
robots object
the selected robot values
template string
the template code of the template to be rendered
uri object
the url in parts
restricted bool
whether this request is restrited or not
error(code, stack) function
renders an error page and ends the request
view() function
renders the selected template with the current context configuration and ends the request
json(data) function
renders the given json object as json and ends the request
html(text) function
renders the given text as html and ends the request
redirect(url, code) function
redirects to the given url with the given code (or 302 as default)
