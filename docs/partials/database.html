database

languages

the languages data collection holds the site languages of the target site.
this collection is used only when the site is configured to be multilingual.

_id objectId
the object identifier
code string
the two letter ISO code of the language
name string
the display name (most commonly used by the admin)
active bool
indicates whether this is an active language or not
default bool
indicates whether this is the default language or not (only one object in the collection should be marked as default)

[nodes table description]

_id objectId
the object identifier
name string
the display name (most commonly used by the admin)
domain string
the domain identifier, typically the module domain in order to group nodes together
entity string
a named entity name identifier, typically the type of the node
parent string
the parent node id (should be empty if the parent is the root)
active bool
indicates whether this is an active language or not
default bool
indicates whether this is the default language or not (only one object in the collection should be marked as default)
