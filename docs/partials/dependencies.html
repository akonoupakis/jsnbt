dependencies

nodeJs

You will use nodeJs as the web server to host your jsnbt site.
Download from http://nodejs.org/download/.

bower

You will be using bower to download any js packages and deploy to your jsnbt site.
Use npm to install: npm install -g bower.

grunt

You will be using grunt to bundle the app source code towards deployment.
Use npm to install: npm install -g grunt-cli.

mongoDB

Your database!
Download from https://www.mongodb.org/downloads.

phantomJs

Phantom is a headless chrome browser.
Since jsnbt works with client side libraries as angular, it needs phantom to prerender its pages if a request comes from a search engine spider bot.
Download from http://phantomjs.org/download.html.

graphics magick

jsnbt uses GraphicsMagick to manipulate uploaded image files.
Download from http://www.graphicsmagick.org/download.html.
