dependencies

NODEJS you will use nodeJs as the web server to host your jsnbt site download from http://nodejs.org/download/

BOWER you will be using bower to download any js packages and deploy to your jsnbt site use npm to install: npm install -g bower

GRUNT you will be using grunt to bundle the app source code towards deployment. use npm to install: npm install -g grunt-cli

MONGODB your database! download from https://www.mongodb.org/downloads

PHANTOMJS phantom is a headless chrome browser. Since jsnbt works with client side libraries as angular, it needs phantom to prerender is pages if a request comes from a search engine spider bot. download from http://phantomjs.org/download.html

GRAPHICSMAGICK the jsnbt uses GraphicsMagick to manipulate uploaded image files. download from http://www.graphicsmagick.org/download.html
